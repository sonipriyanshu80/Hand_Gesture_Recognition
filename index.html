  • Victory (2 fingers)
  • Open Palm (5 fingers)

üöÄ Installation

Requirements: Python 3.7 to 3.12 (MediaPipe doesn't support Python 3.13 yet)

Step 1: Clone the repository

git clone https://github.com/sonipriyanshu80/hand_gesture.git
cd hand_gesture

Step 2: Install dependencies

pip install -r requirements.txt

üíª Usage

Run the program:

python main.py
  • The webcam window will open
  • Show your hand in front of the camera
  • Hand landmarks will be drawn on the screen
  • Gesture name and finger count will be displayed
  • Press 'q' key to quit
